2. Counter

3. Prime Number Checker

This is a slightly useful page that helps you check whether a number is prime or not. It also includes a JavaScript isprime() function that returns true or false.

4. PPP Adjustment Calculator

This program is quite interesting and actually useful. It tells you what a given amount of foreign currency would feel like in your native currency. It also includes a JavaScript function compare() that calculates and displays the direct cost of the foreign currency in your native currency.
